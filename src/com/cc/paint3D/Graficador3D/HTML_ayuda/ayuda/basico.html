Lo Basico

El Simulador de Trayectorias de Persecución de Partículas "STPP" crea escenas en 3D las que pueden ser guardadas como applets y editadas posteriormente. Una Distribucion simple puede consistir de uno o muchos de los componentes de libreria con un comportamiento especificado.

Despues de iniciar el programa "STPP" usted vera dos ventanas. la de arriba es la Barra de herramientas (para crear, editar los componentes de la escena) mientras que la de abajo es el applet actual. la secuencia basica de operaciones para usar el programa es la siguiente:

Seleccione un componente para colocarlo en el applet

Presione el boton de "componente" para seleccionar el componente que quiera. El componente siempre aparece en el centro del applet. Recuerde que usted necesita colocar iluminacion a la escena de alguna manera para que el componente sea visible.

Cambiando las propiedades del componente

Ahora persione el boton de "Propiedades" para seleccionar la propiedad del componente que quiere cambiar. Una descripcion mas amplia se encuentra en propiedades que se describirá mas adelante.

Agregar Comportamiento

El boton permite fijar el comportamineto individual de los componentes . Este solo se puede editar si hay un componente de la escena qu este seleccionado. Para ver la escena final con el comportamiento Ejecutandose usted deberia presionar el boton ejecutar.

Cambiar la posicion y la orientacion de los componentes en la escena

La posicion de los componentes puede ser cambiada usando las tecnicas para el posicionamiento estandard de java3D. Presione el boton derecho del 'mouse' y arrastre mientras presiona para cambiar las coordenadas en 'X' y 'Y'. Presione el boton izquierdo del 'mouse' y arrastre mientras presiona para cambiar la coordenada en 'Z' tambien funciona con las teclas (z,x,c) y -(a,s,d). la orientacion se puede cambiar usando el boton izquierdo del mouse.

Guardar el Grafico en un applet

Cuando tenga el applet de la manera que quiera guardelo. Graficador3D crea un archivo ".HTML" para el applet y un archivo ".JAR" el cual contiene todos los recursos necesarios para ejecutar el applet incluyendo cualquier imagen necesaria. El directorio por defecto donde se crean loa archivos de los applets es el directorio "applets_creados" en el mismo directorio esta el archivo "Graficador3D_applet.jar".
